## Routing rule (why this is not a lane)

A lane needs items an hourly run can advance; "interpret more" is not executable. Threads here earn lane
entries only by producing a fixture spec (→ Lane 1) or a hierarchy definition/vocabulary requirement
(→ Lane 2). The finality-operator question (I-1..I-5) is north-star content: when Hierarchy v0.2's
finality level is drafted, it must engage this register's live rows — that is Lane 2 work, not a third
lane. Guard: creating an interpretations lane would re-run the north-star/byproduct inversion in a new
costume (a shiny thread drawing scheduling weight away from the purpose it belongs to).
