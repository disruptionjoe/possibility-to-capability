# Papers

This directory holds provisional publication-oriented byproducts of the
Possibility to Capability research program.

Paper work is Track 2: valuable in its own right and potentially useful to the
field, the public, or Joe's credibility, while subordinate to the Track-1
force-or-falsify North Star. Every candidate must state which North-Star test,
result, method, or negative finding produced it.

Nothing here is externally published or promoted by placement. External
publication, posting, submission, or outreach requires Joe's explicit
authorization and all applicable source-repository gates.

- `candidates/` contains internal paper candidates and seeds.
